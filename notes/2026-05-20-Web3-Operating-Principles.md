## 交易到出块的完整链路

```
Wallet → RPC/Node → Mempool → Builder → Validator → Block
```

| 环节 | 说明 |
|------|------|
| **Wallet** | 在本地签名交易，**私钥不发出** |
| **RPC/Node** | 接收并传播交易到网络 |
| **Mempool** | 尚未被打包的交易临时排队区 |
| **Builder / Validator** | 选择 / 排序 / 提议 / 证明区块 |
| **Block / Explorer** | 交易写入区块后，链上可公开查询 |

> 📺 **实况演示：** Bruce 打开 Etherscan 展示了真实区块数据，包括 Gas Price、区块大小、Builder 信息等。

## 共识机制：PoW vs PoS

| 维度 | PoW（BTC） | PoS（ETH） |
|------|-----------|-----------|
| 核心机制 | 算力竞争写账 | 验证者质押 ETH |
| 出块方式 | 解一道"很难但答案很好验"的题 | 系统伪随机选取 Proposer 和 Validator |
| 惩罚 | 无（算力浪费） | 作恶或严重失职会被**罚没（Slash）** |

---

## 最终确认性（Finality）

- Ethereum PoS 中，正常情况下约 **2 个 epoch**（12.8 分钟）达到 finalized
- **川补："超过 12 分钟就非常安全了"** 🎯
- 例外情况：网络异常、客户端问题、验证者参与不足时 finality 可能延迟

> 区块像账本新一页，会引用上一页指纹（hash）；后续区块越多，推翻历史越难。

---

## 节点客户端多样性

- 节点软件不止一种：**执行层客户端 + 共识层客户端**
- 多样性越高，越不怕单一软件 bug 或分叉 finality 风险
- 去中心化不仅是"节点多"，还包括**实现多样"**
- 以太坊历史上出现过客户端 bug 导致 finality 异常的案例

---

## 钱包 / RPC / 节点 / 网络：如何链接

- 钱包通常不直连全网，它连接某个 **RPC**（访问节点的 API）
- RPC 背后是节点或节点集群
- **入口层中心化风险：** 很多用户依赖少数 RPC 服务商 → 可用性、隐私、审查、单点故障风险
- 这不等于 Ethereum 共识层中心化，但入口层确实存在瓶颈

## 智能合约

### 是什么

> 写在链上、能被交易触发的代码。

- 不是真的"有智能"，只是按代码规则执行
- 在 **EVM**（以太坊虚拟机）里运行
- 交易触发执行 → 改变链上状态 → 写入区块后可查

### 社会学意义

| 特性 | 说明 |
|------|------|
| **规则能看见** | 合约代码、调用记录和执行历史都可检查 |
| **少一点临时拍板** | 条件满足后按代码执行，临时改规则的空间变小 |
| **信任对象变了** | 从"相信某个机构"转向"检查代码 + 共识 + 审计/治理" |

### 代理合约（Upgradeable Contract）

- **原始合约不能变，但可以做升级**
- 通过 Proxy Contract → `delegatecall` → Implementation Contract 模式实现
- 这是以太坊上智能合约升级的通用做法

## Web3 关键特性

### （一）去中心化

- 钱包创建高度去中心化
- 交易广播、RPC、前端入口都可能被审查
- 节点全球分布、入口可替换，但入口层仍可能有中心化风险

### （二）其他特性

| 特性 | 说明 |
|------|------|
| **开放开源 / 可验证** | 客户端开源，交易记录公开可查询，规则与执行可审计 |
| **隐私边界** | 主流公链默认是**公开账本 + 伪匿名**，不是默认隐私系统 |
| **可组合性** | 公开合约和标准接口让应用像乐高一样互相调用，但也放大级联风险 |

## 核心框架：Web3 是三门学科的交叉

```
Web3 = 密码学 + 经济学 + 社会学
```

| 学科 | 作用 |
|------|------|
| 🔐 **密码学** | 哈希、ZK（零知识证明）让权利和行为可以被验证 |
| 💰 **经济学** | Gas、质押和罚没把成本、奖励写进系统规则里 |
| 👥 **社会学** | 个人主权、去中心化治理和共识形成，回答的是"谁说了算" |

> **Web3 的核心不是技术本身，而是数字世界控制权的重新分配。**
> 它用密码学保障所有权，用经济激励协调参与者，用社会共识决定规则。
