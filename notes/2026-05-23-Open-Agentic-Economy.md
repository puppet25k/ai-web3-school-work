## 核心论点

> **Ethereum is the decentralized settlement and coordination backbone for machines in the machine economy, with humans at the center.**

- 一个中立层，机器可以在其中大规模协调，同时人们保留自主权、所有权和控制权
- AI Agent 在其上交易的基础设施，将决定这个新经济的运作方式：
  - 谁能参与
  - 信任如何建立
  - 价值流向哪里
  - 竞争环境是否开放

## ERC-8004: Agent 身份与信任

> **Created:** August 2025 · **Live on mainnet:** January 2026

一个给 AI Agent 提供可验证身份和公开信誉记录的协议标准。

### 三大注册表

| 注册表 | 功能 |
|--------|------|
| **Identity Registry** | Agent 身份注册 |
| **Reputation Registry** | 信誉记录 |
| **Validation Registry** | 验证机制 |

- 作者：Marco De Rossi, dcrapis
- 让 Agent 能确切知道是否可以信任某个软件

---

## X402: 机器对机器支付

- 复兴 HTTP **402 "Payment Required"** 状态码
- 为 API 和 Agent 提供标准化的定价和链上支付证明方式
- 让 Agent 之间的支付无需信用卡或 API Key
